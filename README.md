# IBDP Chemistry Interactive Organic Handbook (First Assessment 2025)

An interactive digital handbook for the International Baccalaureate Diploma Programme (IBDP) Chemistry curriculum.

## Features

- Interactive textbook
- Student Mode
- Teacher Mode
- SVG reaction mechanisms
- Organic reaction explorer
- Functional group explorer
- Search
- Dark mode
- Offline support
- Printable notes
- Interactive quizzes

## Coverage

- Reactivity 3.3 – Electron Sharing Reactions
- Reactivity 3.4 – Electron-Pair Sharing Reactions
- Reactivity 3.2 – Organic Redox
- Structure 3.2 – Functional Groups & Spectroscopy
- Structure 2.4 – Polymers
